Role of metadata catalog:
    * Discovery: Enables identifying and locating datasets across the distributed 
    environment.
    * Data Referencing: Stores "data_path" links pointing to the physical location 
    of datasets at provider nodes.
    * Decoupling: Separates the process of searching for data from the process of 
    storing it.

Discussion Questions:
    * Which components store the actual data?
    The independent data providers.

    * Which component enables dataset discovery?
    The federated metadata catalog.

    * What advantages does the federated architecture provide?
    It enables collaboration through distributed providers and shared metadata 
    without requiring central data aggregation.

    * What limitations might such a system have?
    The system relies on metadata accuracy; if a provider changes a local path 
    without updating the catalog, the data becomes unreachable.
